# Manage your user profile
You can manage and customize your user profile via the account menu.
[![User account menu location](/docsassets/documentation/manage-your-user-profile/UserAccountMenu.png)](/docsassets/documentation/manage-your-user-profile/UserAccountMenu.png)
Using the menu, you can:
  * View and edit your account details by selecting **Account**. 
    * On the **General** tab, you can set the basic details of your profile. 
      * **Admin UI language** â€“ Select your preferred language for the administration interface. This option is only available if your system administrator has [configured](/documentation/developers-and-admins/customization/admin-ui-localization) additional languages in the system.
    * On the **Password** tab, you can change the password you use to sign in to the Xperience administration.
  * Sign out of the administration by selecting **Sign out**.
