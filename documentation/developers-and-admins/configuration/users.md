# Users
### [User management](/documentation/developers-and-admins/configuration/users/user-management)
Learn how to invite users to the Xperience administration interface and manage their accounts in the system.
### [Role management](/documentation/developers-and-admins/configuration/users/role-management)
The system provides a role-based access control system for determining access privileges to various administration interface features and applications. Learn how to define, manage, and assign roles to users in the system.
### [Administration - Registration and authentication](/documentation/developers-and-admins/configuration/users/administration-registration-and-authentication)
Xperience provides an integration API based on [ASP.NET Identity](https://github.com/dotnet/aspnetcore/tree/master/src/Identity) used when managing users with access to the Xperience administration.
### [Notifications](/documentation/developers-and-admins/configuration/notifications)
Learn how to create and edit notification emails, which provide information to administration users about various events in the system.
