# Learn more about Data protection
You’ve reached the end of the **Data protection** module.
To recap, the examples throughout the module have illustrated the following key topics:
  * Creating a mapping between consents and cookie levels
  * Building user interfaces for visitors to manage their consent
  * Conditionally displaying certain content based on whether a visitor has provided consent
  * Handling visitor requests to collect or erase their personal data
