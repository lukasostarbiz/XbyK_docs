# Work with widgets
In this video tutorial, you’ll learn about the basic operations with widgets.
## Prerequisites
  * You should know how to [create a page](/documentation/business-users/website-content/create-pages) in Xperience by Kentico.


## What you will learn
  * How to add different widgets.
  * How to add content to widgets.
  * How to personalize widgets.
  * How to move, copy, and paste widgets.
  * How to remove widgets.


### Next step
Now that you know how to work with the Page Builder, learn how to create specific pages. Let’s start with [a new article website](/guides/digital-marketing/work-with-websites/create-new-article-on-a-website).
You can also learn more about the [Page Builder](/guides/digital-marketing/xperience-essentials/manage-website-content-essentials#page-builder-in-the-website-channel) in the Tutorial.
