Let’s create a new user account for a new Data Protection officer that just joined your organization.
  1. Navigate to the **Users** application.
  2. Select **New user**.
  3. Add information about the user: 
    1. **New user email** : e.g., gregm@dancinggoat.local,
    2. **Role** : Data Protection Officer.
  4. **Send** the invitation.
  5. Open the newly created user.
  6. Add: 
    1. **First name** : e.g., Gregory,
    2. **Last name** : e.g., Matthews.
  7. **Save** the changes.
