# Update user information
You need to be an **Administrator** to update other usersâ€™ information. 
To update information of other users:
  1. Go to the **Users** application.
  2. Open the user whose information you want to edit.
  3. Update the fields as needed: 
    1. **First name** : e.g., _Laura_.
    2. **Last name** : e.g., _Miller_.
  4. **Save** the changes.
