Not a fan of the interactive video format? Here’re the steps you need to take to add users to a role in Xperience by Kentico.
  1. Go to the **Role management** application.
  2. Select the role you want to add users to.
  3. Go to the **Assigned users** tab on the left. You can see all users with the particular role.
  4. You can add more users to this role by selecting **Add user**.
