# Update role permissions
  1. Go to the **Role management** application.
  2. Select the role you want to change.
  3. Switch to the **Permissions** tab and update the permissions.
  4. In the drop-down list, select the application you want your user to have access to and choose which actions the user can perform.
  5. **Save** your changes.
