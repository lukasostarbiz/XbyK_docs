# Find data Xperience collected about a contact
To find contact data collected by Xperience:
  1. Navigate to the **Data protection** application and open the **Right to access** tab.
  2. Insert the contact’s email and use the **Search** button.
  3. Xperience displays the contact’s online marketing data.
